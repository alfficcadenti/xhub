## Development

1. To start developing locally, clone the project:

   ```bash
   git clone git@github.expedia.biz:EGPlatform/opxhub-ui.git
   ```

2. Install the dependencies:

   ```bash
   npm install
   ```

3. Run the application

   ```bash
   npm start
   ```

4. Navigate to `localhost:8080`
